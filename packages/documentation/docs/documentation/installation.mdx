# Installation

## Via Package Manager

Install the package using your preferred package manager:

<Tabs
  defaultValue="npm"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Bun', value: 'bun'},
    {label: 'Yarn', value: 'yarn'},
  ]}>
  <TabItem value="yarn">
    <CodeBlock className="language-shell">yarn add react-hotkeys-hook</CodeBlock>
  </TabItem>
  <TabItem value="npm">
    <CodeBlock className="language-shell">npm i react-hotkeys-hook</CodeBlock>
  </TabItem>
  <TabItem value="bun">
      <CodeBlock className="language-shell">bun i react-hotkeys-hook</CodeBlock>
    </TabItem>
</Tabs>

```jsx
import { useHotkeys } from 'react-hotkeys-hook';
```

## Via CDN

You can import the ES Module directly from [unpkg.com](https://unpkg.com):

```js
import { useHotkeys } from 'https://unpkg.com/react-hotkeys-hook@latest/dist/react-hotkeys-hook.esm.js';
```
